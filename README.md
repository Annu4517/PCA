## Description

This project demonstrates the implementation of Principal Component Analysis (PCA) to reduce the dimensionality of a dataset while preserving the maximum amount of information (variance). PCA is an important technique used in data preprocessing, visualization, and improving machine learning model performance.

The project is implemented using Python in a Jupyter Notebook
---------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Objectives
---------------------------------------------------------------------------------------------------------------------------------------------------------------------
Understand dimensionality reduction

Apply PCA on a dataset

Reduce feature space efficiently

Analyze explained variance

Visualize transformed data
